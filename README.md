# Easy Factorization

这是一个可以帮助你进行因式分解的小程序。

## 部署方法

从release中下载压缩包后解压，运行“启动.bat”

从源代码运行：

```bash
git clone git@github.com:Xdyu1522/easy_factorization.git
cd ./easy_factorization
pip install -r requirements.txt
python ./easy_factorization.py
```
